## Viewing Data

Use **DataFrame.head()** and **DataFrame.tail()** to view the top and bottom rows of the frame respectively:

```
df.head()
df.tail()
```

Display the **DataFrame.index** or **DataFrame.columns**:

```
df.index
df.columns
```

**describe()** shows a quick statistic summary of your data:
```
df.describe()
```
